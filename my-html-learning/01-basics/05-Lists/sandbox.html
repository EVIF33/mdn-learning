Three Types of HTML Lists

1. <ul> — Unordered List (bullets)

When: order doesn't matter (shopping list)

Example:

  • Milk
  • Eggs
  • Bread
  • Hummus

2. <ol> — Ordered List (numbers)

When: order matters (instructions, recipes)

Example:

  1. Drive to the end of the road
  2. Turn right
  3. Go straight through two roundabouts
  4. Turn left at the third roundabout

3. <dl> — Description List (terms)

When: terms and their definitions

Example:

Soliloquy
In drama, when a character speaks to themselves, expressing inner thoughts.
Monologue
When a character speaks their thoughts aloud for the audience.
Aside
A comment only for the audience (humor or drama).
Also: additional information alongside main content.

4. Nested Lists

You can nest one list inside another

  1. Remove garlic skin, chop coarsely
  2. Remove pepper seeds, chop coarsely
  3. Add all ingredients to food processor
  4. Process into paste:
    • For coarse "chunky" hummus — process briefly
    • For smooth hummus — process longer

Important: Close nested list before closing parent <li>
